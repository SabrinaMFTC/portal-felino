## 3. Ideação

Este projeto consiste em um site dinâmico, que contém diversas informações úteis para tutores de gatos e pessoas que tenham interesse em adotar gatos, tais como:

- ONGs para adoção de gatos;
- Pontos de doação de ração;
- Pontos de doação de medicamentos;
- Pontos de castração;
- Exigências para adoção;
- Cuidados específicos de cada raça;
- Indicações de cat sitter.

O caráter extensionista do projeto consiste no impacto real que a construção de um site tal qual o idealizado pode gerar, tendo em vista, por um lado, o exponencial crescimento no interesse na adoção de pets, em particular gatos, desde o advento da pandemia, e, por outro lado, a falta de informação centralizada e confiável a respeito de cuidados necessários e como adotar um gato.
