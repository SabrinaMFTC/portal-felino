## 3. Ideação

Este projeto consiste em um site dinâmico que reúne diversas informações úteis para tutores de gatos e pessoas interessadas em adotar gatos, tais como:

- ONGs para adoção de gatos;
- Pontos de doação de ração;
- Pontos de doação de medicamentos;
- Pontos de castração;
- Exigências para adoção;
- Cuidados específicos de cada raça;
- Indicações de cat sitter.

O caráter extensionista do projeto está relacionado ao impacto real que a construção de um site tal qual o idealizado pode gerar. Isso se justifica, por um lado, pelo crescimento expressivo do interesse pela adoção de pets - em especial gatos - desde o advento da pandemia e, por outro lado, pela ausência de informações centralizadas e confiáveis acerca dos cuidados necessários e do processo de adoção.
